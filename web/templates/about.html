{% trans 'About us' %}

{% blocktrans %}We are two developers and relatively casual players of the game. It's the first project we made together.{% endblocktrans %}

{% blocktrans %}We're both 23 years old and we live in San Francisco, California. We come from the same computer science university and both work as software engineers in the same company we founded:{% endblocktrans %}Rainbow Cookies. {% blocktrans %}Though we love our job, we don't pay ourselves and don't have any income, so{% endblocktrans %} {% trans 'any donation' %} {% trans 'helps a lot and is greatly appreciated.' %}

{% blocktrans %}The heartwarming support from the community keeps motivating us to do more! We love helping the community by providing as much information as we can about the game, while keeping the website easy to use, pink and cute ^_^{% endblocktrans %}

{% blocktrans %}We don't know how far we will go with this, but one thing we know for sure is that we are having a lot of fun doing it and sharing this crazy idol adventure with you.{% endblocktrans %}
